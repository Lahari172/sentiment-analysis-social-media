# Sentiment Analysis on Social Media

## Overview

This NLP project classifies social media posts into different sentiment categories using Machine Learning.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression
- Google Colab

## Features

- Text preprocessing
- Sentiment classification
- Model evaluation
- Prediction on custom text

## Result

Achieved approximately 80–90% accuracy on the test dataset.
